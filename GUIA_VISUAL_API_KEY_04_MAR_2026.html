Encontrar y Editar la API Key

3

Agregar tapati.online a las Restricciones

Una vez dentro de la edición de la API Key:

  1. Baja hasta la sección "Application restrictions"
  2. Verás seleccionado: "HTTP referrers (web sites)"
  3. En "Website restrictions", verás una lista como:
Lista actual de dominios:
Website restrictions
localhost/*
citard-fbc26.web.app/*
citard-fbc26.firebaseapp.com/*
+ ADD AN ITEM

Ahora agrega los nuevos dominios:

  1. Haz click en "+ ADD AN ITEM"
  2. Escribe: tapati.online/*
  3. Haz click en "+ ADD AN ITEM" de nuevo
  4. Escribe: *.tapati.online/*
  5. Haz click en el botón "SAVE" (azul, abajo)
Configuración final esperada:
