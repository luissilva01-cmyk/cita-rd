🎨 Ta' Pa' Ti - Logos

Elige el diseño perfecto para tu app de citas

⭐ RECOMENDADA Versión 2 - Compacta

Logo limpio y moderno. Perfecto para iconos de app, favicon y uso general. Basado en la segunda imagen que compartiste.

Ventajas

  • Más compacto
  • Mejor en tamaños pequeños
  • Moderno y limpio
  • Versátil

Consideraciones

  • Menos impacto visual
  • Requiere buen contraste
Uso en React:
<Logo size={60} variant="color" version="v2" />
ALTERNATIVA Versión 1 - Grande

Logo más grande con mayor presencia visual. Ideal para landing page y marketing. Basado en la primera imagen que compartiste.

Ventajas

  • Mayor impacto visual
  • Perfecto para hero sections
  • Más detallado

Consideraciones

  • Ocupa más espacio
  • No ideal para favicon
  • Menos versátil
Uso en React:
<Logo size={80} variant="color" version="v1" />
MINIMALISTA Versión Simple

Diseño ultra minimalista con formas geométricas básicas. Máxima simplicidad.

Ventajas

  • Extremadamente simple
  • Carga rápida
  • Escalable

Consideraciones

  • Menos personalidad
  • Puede parecer genérico
Uso en React:
<Logo size={50} variant="color" version="simple" />

📖 Guía de Uso

Cómo usar en tu código

1. Importar el componente

import Logo from './components/Logo';

2. Usar en tu JSX

// Logo por defecto (v2, color, 40px) <Logo /> // Logo grande para landing <Logo size={80} variant="color" version="v2" /> // Logo blanco para navbar oscuro <Logo size={50} variant="white" version="v2" /> // Logo con texto debajo <Logo size={100} variant="color" version="v2" showText={true} /> // Logo versión 1 para hero section <Logo size={120} variant="color" version="v1" />

3. Props disponibles

Prop Tipo Default Descripción
size number 40 Tamaño en píxeles
variant 'color' | 'white' | 'black' 'color' Variante de color
version 'v1' | 'v2' | 'simple' 'v2' Versión del diseño
showText boolean false Mostrar "Ta' Pa' Ti" debajo
className string '' Clases CSS adicionales

💡 Recomendación

Te recomiendo usar la Versión 2 (Compacta) como logo principal. Es más versátil, funciona mejor en tamaños pequeños (favicon, iconos de app), y tiene un diseño más moderno y limpio. Puedes usar la Versión 1 para la landing page si quieres más impacto visual en el hero section.
