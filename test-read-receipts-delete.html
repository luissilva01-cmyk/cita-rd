📖 Instrucciones

✅ Mensajes Vistos:

  1. Usuario A envía un mensaje → Verás un check gris (✓)
  2. Usuario B abre el chat (automático) → El check se vuelve azul (✓✓)
  3. Esto indica que el mensaje fue leído

🗑️ Borrar Mensajes:

  1. Haz click derecho en cualquier mensaje
  2. Selecciona "Borrar para mí" → Solo tú dejas de verlo
  3. Selecciona "Borrar para todos" (solo tus mensajes) → Todos dejan de verlo
  4. Selecciona "Copiar mensaje" → Se copia al portapapeles

📱 En Móvil:

Mantén presionado el mensaje por 500ms para abrir el menú contextual
