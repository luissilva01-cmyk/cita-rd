## 🔧 SI QUIERES CAMBIAR EL LOGO

Si el logo actual no te gusta o quieres uno diferente:

### Opción 1: Usar otra versión
Edita `cita-rd/views/views/Landing.tsx` línea 83:

```tsx
// Versión actual (v2 - compacta)
<Logo size={80} variant="color" version="v2" />

// Cambiar a v1 (más grande)
<Logo size={80} variant="color" version="v1" />

// Cambiar a simple (minimalista)
<Logo size={80} variant="color" version="simple" />
```

### Opción 2: Contratar diseñador profesional
Lee el archivo `ACCION_INMEDIATA_LOGO.md` para contratar un diseñador en Fiverr ($25-50, 3-5 días).
